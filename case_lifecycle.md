# Full Case Lifecycle — Beacon of Hope

## Comprehensive Flowchart


## Summary: Key Data Points Per Phase

| Phase | Primary Table(s) | Key Fields |
|-------|-----------------|------------|
| 1. Triggering Event | `residents` | `case_category`, `sub_cat_*` (10 boolean flags) |
| 2. Referral & Intake | `residents` | `referral_source`, `referring_agency_person`, `initial_risk_level`, `initial_case_assessment`, `date_of_admission`, `assigned_social_worker` |
| 3. Safehouse Placement | `residents` + `safehouses` | `safehouse_id`, `date_enrolled`, `date_case_study_prepared` |
| 4. Ongoing Care | `process_recordings`, `intervention_plans`, `education_records`, `health_wellbeing_records`, `incident_reports` | All fields — this is the bulk of the data |
| 5. Home Visitations | `home_visitations` | `visit_type`, `family_cooperation_level`, `safety_concerns_noted`, `visit_outcome` |
| 6. Reintegration Planning | `residents` | `reintegration_type`, `reintegration_status`, `current_risk_level` |
| 7. Case Closure | `residents` | `case_status`, `date_closed` |
| 8. Post-Placement | `home_visitations` | `visit_type = 'Post-Placement Monitoring'` |
